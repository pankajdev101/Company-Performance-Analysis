# Future 500 SQL Analysis Project

## Project Overview
This project analyzes the Future 500 dataset using SQL Server. The goal is to identify business insights related to revenue, profit, growth, industries, and states.

## Tools Used
- SQL Server
- Excel (Dataset Source)
- Power Bi


## Dataset Columns
- ID
- Name
- Industry
- Inception
- Employees
- State
- City
- Revenue
- Expenses
- Profit
- Growth

## Data Cleaning
- Removed duplicate records based on ID
- Handled NULL values
- Converted columns to appropriate data types
- Cleaned Expenses column

## SQL Analysis Performed

### Revenue Analysis
- Calculated total revenue
- Identified top 10 companies by revenue

### Profit Analysis
- Calculated total profit
- Identified top profitable companies

### Industry Analysis
- Industry-wise revenue
- Industry-wise profit
- Average growth by industry

### State Analysis
- State-wise revenue
- State-wise profit

### Employee Analysis
- Average employees per company
- Revenue per employee

### Growth Analysis
- Companies with highest growth
- Industry-wise growth trends

## Key Insights
- Identified top-performing companies based on revenue and profit
- Found industries generating the highest revenue
- Analyzed growth trends across industries
- Compared company performance across different states
